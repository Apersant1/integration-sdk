## Установка библиотеки

Для установки библиотеки `@infomaximum/integration-sdk` в ваш проект используйте один из следующих способов:

### Используя npm:

```bash
npm install @infomaximum/integration-sdk
```

### Используя yarn:

```bash
yarn add @infomaximum/integration-sdk
```
